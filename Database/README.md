# Database

se incluye la DB en formato SQL, **tu_mercado_sena** esta es una copia de la migración creada con la API de Laravel, por lo que si puede optar por hacer la migración, es lo recomendable

incluye una imágen del modelo **MR** sacado de la DB

hay un software en Python (**PoblarDB**) para poblar la DB con miles de datos sintéticos aleatorios

la carpeta **TMS_API** es una API sin framework, sencilla, para manejar la DB, utilizada para funciones administrativas del entorno Desktop
